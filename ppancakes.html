perfect pancakes

summary

For my UI/UX course, we were tasked to create a live-filtering page for any content type of our choice, using React.

Perfect Pancakes was inspired by a randomly generated Github repository url, 'perfect-pancake'. I took that inspiration, made it plural, and decided to filter different international pancakes. You can view the final website for Perfect Pancakes here!

content used (credit where due)

All pancake information was taken from this Spoon University article, and I chose to find and source all the pictures used myself - all the picture sources can be found in the Github repository for this project.

step 1: react components

Since the purpose of this project was to learn React, I used React Components when figuring out how to organize my site. Each pancake 'card' is its own react component, as is the list that sorts and displays them.

step two: final site

The final iteration of this site has a sortable list of pancakes. You have the option to filter by the continent of origin, or the 'taste' of the pancake (this still needs work, as some pancakes can be both sweet and savory). I also added an option for alphabetical sorting, as part of extending the functionality of the site.

reflection

I appreciated this project, as I had been meaning to try and learn React for a couple years now. I also liked that I could goof off a bit and make a site that was guaranteed to make everyone who looked at it very, very hungry (including me...). I didn't quite get to improve the styling of the site as much as I would like, because I spent so much time on the actual functionality. In the future, I'd really like to make the site and the button styling a little more appealing and reminscent of pancakes.
